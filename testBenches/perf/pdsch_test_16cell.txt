    --g                    Group together all PUSCH cells in a context in a slot and execute via single PUSCH pipeline object; same for PUCCH and PRACH if present

    --c <ch_name_1,...,ch_name_N> enable reference checks for channels whose names are provided as a comma separated list

    -n                     Use green contexts
